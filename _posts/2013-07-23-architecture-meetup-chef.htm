Architecture Meetup — Chef (Tuensday, 23rd July 2013 - 6:30 for 7pm)

At the next Software Architecture meetup, Adrian Moisey will be presenting an introduction to using Chef to manage your systems. Topics covered include:

1. An overview, and getting started
2. The various ways Chef can run
3. Best practices for using roles and environments
4. Things you should and should not do

If we have time afterwards, there will be a brief discussion of/comparison to other configuration management systems.

Date: July 23, 2013
Time: 6:30 for 7pm
Location: codebridge - http://www.codebridge.co.za/directions.htm

We don't have a beer sponsor yet, but there will be beer.
